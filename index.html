Como configurar
  1. Acesse supabase.com e crie uma conta gratuita
  2. Clique em New project, escolha um nome e defina uma senha
  3. Aguarde o projeto inicializar (1-2 minutos)
  4. No menu lateral, vá em SQL Editor e cole o conteúdo do arquivo setup.sql que acompanha esta aplicação
  5. Execute o script (botão Run). Isso cria as tabelas e dados de exemplo
  6. Vá em Project Settings → API, copie a Project URL e a anon public key
  7. Cole aqui acima e clique em Salvar e conectar
